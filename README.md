# G+Don

G+Don は Mastodon のためのウェブアプリです。
利用するにはどこかのMastodonインスタンスにユーザー登録している必要があります。

## Development setup


```npm install```


```npm run dev```


**Using library**

***Server side***

 * Node.js
 * express
 * nunjucks

 
***Client side***

 * Vue.js
 * Vuetify
 * vue-carousel
 * Materialize
 * jQuery
 * jquery-timeago
 * lodash
 * alertify
 * paste
 * mastodon
 * twitter-text.js -> modified to mastodon-text.js
